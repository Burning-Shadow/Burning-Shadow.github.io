Webpack原理初探

Posted on 2019-08-05 | In 打包工具

​ 作为最常用的打包工具之一 webpack 困扰也哺育着成千上万的前端开发，我们都知道 input output，modules plugins loader，可是站在更高一层的角度看这些都有什么区别？什么作用？实现原理是 js 中的哪一种数据结构？恐怕很少有人去考量。所以今天我们试着一探究竟。文末我会试着根据 vue-cli 脚手架的配置项来解读 webpack 的作用与功能

是什么

初学 webpack 的小伙伴想必和我有一样的问题：webpack 是什么？

​ 简而言之，webpack 算是一个继承打包、压缩功能的前端编译工具。而真正将其 引入历史舞台的原因，引用表严肃表哥的一句话：浏览器实现之初没有做到模块化，之后都是在补作业 。所以 webpack 的作用我们就可以理解为：使前端页面得以通过模块化加载方式进行载入。

​ 当然，打包压缩什么的算是后话，原理暂时这么记。
