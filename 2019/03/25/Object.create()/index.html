可见 ES6 中的类继承具备以下几个特点：

  • 处于严格模式下，给各种属性变量多一层保障（防止变量提升）
  • 子类通过原型链继承继承父类的方法，通过构造函数式继承继承父类的属性（防止子类实例间相互污染）
